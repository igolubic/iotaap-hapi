# IoTaaP HAPI - Hardware API Library
> Hardware to Cloud Application Programming Interface

## Usage

Include `IoTaaP_HAPI.h` in your project.

## Usage example

```
#include "IoTaaP_HAPI.h"

IoTaaP_HAPI iotaap_hapi;
```
